About Fabric Defect Classification

Project Overview

Fabric Defect Classification is an AI-powered system designed to automatically detect and classify various types of defects in Fabric Fabrics. This system aims to improve quality control in Fabric manufacturing by providing fast and accurate defect identification.

Technology Stack

  • Deep Learning: Using a MobileNetV2-based convolutional neural network for image classification
  • TensorFlow: Framework used for model training and inference
  • Flask: Web framework for the user interface
  • Bootstrap: Front-end component library for responsive design
  • Chart.js: JavaScript library for visualizing prediction probabilities

Defect Types

Defect Free

Normal Fabric with no visible defects.

Hole

Physical gaps or perforations in the Fabric.

Horizontal

Horizontal creases or folds in the Fabric.

Lines

Linear marks across the Fabric surface.

Stain

Discoloration or spots on the Fabric.

Vertical

Vertical creases or folds in the Fabric.
